# 29.7. Промежуточная сдача проекта (HW-03)

## Задание 29.3.8

Теперь давайте немного изменим обработку сессии.

1. Сделайте так, чтобы наша функция обработчик сессии — (``` handleSession ```) — 
была объявлена через ``` declaration ``` и возвращала объект сессии.
2. Вынесите консольный вывод информации о сессии в отдельную функцию ``` sessionLog ```. 
Её объявите через ``` expression ```, и вызывайте после ``` handleSession ```. 
``` sessionLog ``` должна принимать объект сессии, который нам вернула ``` handleSession ```.
