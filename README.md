# 29.7. Промежуточная сдача проекта (HW-03)

## Задание 29.6.7

Допустим, мы хотим увеличить аудиторию нашего сайта.
Можно начать с того, чтобы предложить пользователям подписаться на нас в соцсетях.
А чтобы это не раздражало, сделаем это не сразу, а через минуту после входа на сайт. 

Покажите пользователю всплывающее окно с предложением подписаться на наш Инстаграм
спустя минуту после того, как он зайдет на сайт.
