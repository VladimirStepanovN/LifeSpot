# 29.7. Промежуточная сдача проекта (HW-03)

## Задание 28.5.6

Мы разобрались, как сканировать элементы на странице и захватывать пользовательский ввод.
Теперь последний этап — фильтрация содержимого страницы. 

Для этого вам потребуется вспомнить некоторые свойства элементов из курса CSS:
здесь мы будем скрывать ненужные элементы на странице, проставляя им соответствующие CSS-свойства.

**Измените ваш код так, чтобы те элементы, описания которых не содержат того,
что вводит пользователь, скрывались через CSS-свойство ``` display: none ```, а там,
где содержат — показывались (``` display: inline-block ```).**
