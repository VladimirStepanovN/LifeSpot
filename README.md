# 29.7. Промежуточная сдача проекта (HW-03)

## Задание 28.5.5

Захват пользовательского ввода у нас готов. 
Теперь подумаем, как можно фильтровать содержимое на основании ввода пользователя. 

Очевидно, что при каждом изменении значения ввода нам нужно пробежаться по всем элементам и посмотреть,
удовлетворяют ли они запросу. Если элемент удовлетворяет условию, то оставим его видимым, если нет — скроем. 

Но это потом, а пока — промежуточный этап, и здесь нам помогут циклы.

**Напишите код, который будет при вводе/удалении текста в input пробегаться по всем элементам,
содержащим видеотрансляции, и выводить их в консоль.**
