# 29.7. Промежуточная сдача проекта (HW-03)

## Задание 29.5.5

Теперь давайте переработаем нашу функцию ``` handleSession(session) ``` с использованием замыкания. 

Здесь мы видим, что она выполняет две задачи (создает сессию и проверяет на возраст).
Это идёт вразрез с принципом единства ответственности, который вам должен быть знаком из курса С#. 

Переработайте её код так, чтобы было две функции:

* ``` checkAge() ``` для проверки и сохранения возраста;  
* ``` handleSession() ``` для сохранения в сессию данных о времени посещения, а также строки ``` userAgent ```.

Функции при этом не должны принимать параметры, а переменная ``` session ``` будет «захватываться» ими извне.

Также вам предстоит исправить функцию ``` sessionLog(session) ```.
